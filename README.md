# Portfólio Contemporâneo Baseado em Dados - Luciano P. de Moura

Repositório estruturado contendo os arquivos de front-end estático para o portfólio profissional de **Luciano P. de Moura**, projetado com um conceito visual **Futurista Realista (*Data-Driven*)** e hospedado nativamente através do GitHub Pages sob o domínio corporativo do ecossistema Cognivus.

## 🎯 Objetivo do Projeto
O objetivo deste portfólio é unificar e ressignificar uma trajetória profissional altamente multidisciplinar, criando uma ponte sólida entre a sólida experiência em **Administração, PCP e Auditoria de Qualidade** e o momento atual voltado ao **Direito, Jurimetria e Desenvolvimento de Software Assistido por Inteligência Artificial (LexOS)**. 

Serve como plataforma central de autoridade digital em paralelo ao início do programa de estágio estratégico na **Procuradoria Geral do Estado (PGE)**.

## 🛠️ Arquitetura Tecnológica
Para garantir custos de infraestrutura nulos, velocidade de carregamento de alta performance e segurança rígida, o projeto utiliza a filosofia **Jamstack**:
* **HTML5 Estrutural:** Tags semânticas limpas para otimização SEO e acessibilidade.
* **CSS3 Customizado:** Estilização baseada em variáveis nativas, efeitos de iluminação modernos de grade e responsividade total (*mobile-first*).
* **JavaScript Assíncrono:** Lógica leve para componentes de interface dinâmicos (como o efeito *typewriter* de digitação do cabeçalho) sem dependência de bibliotecas externas pesadas.

## 🛡️ Governança de Dados, Compliance e Segurança
Alinhado com as exigências de órgãos institucionais de alta relevância como a PGE, o código-fonte e o escopo visual trazem:
* **Conformidade LGPD:** Arquitetura estática pura que não coleta cookies invasivos ou dados de rastreio de terceiros sem consentimento.
* **Privacy by Design:** Princípios de engenharia contemporânea aplicados para assegurar que soluções derivadas (como o ecossistema LexOS) tratem dados públicos e privados com estrita higienização algorítmica.
* **Segurança Nativa:** Implantação e tráfego criptografado de ponta a ponta via SSL/TLS fornecido automaticamente pelo servidor do GitHub Pages (HTTPS ativo).
